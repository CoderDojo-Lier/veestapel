# About
**veestapel** is an animal-stacker game where 2 players can drop an animal on a platform, turn by turn.

A player loses if an animal falls off the platform during their turn.

# Run
The regular [Scratch editor](https://scratch.mit.edu) has no built-in physics engine. Therefore, [Turbowarp](https://turbowarp.org) is used instead.

- Go to [turbowarp.org](https://turbowarp.org)
- Click **File > Load from your computer...**
- Select **veestapel.sb3**

# Controls


- Player 1 (left):
  - **a**: steer left
  - **z**: steer right
  - **e**: drop animal

- Player 2 (right):
  - **←**: steer left
  - **→**: steer right
  - **SPACE**: drop animal

- General:
  - **r**: restart game
  - **c**: switch between realistic and cartoon-style animals

To adapt the controls, change the code in the *engine* sprite (steering), *grabber* sprite (dropping), and *stage* (restart).
